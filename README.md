# 26 Years of NHS England Hospital Episode Statistics — Interactive Data Visualisation

## Overview
This project analyses 26 years of NHS England Hospital Episode Statistics (HES) data through interactive and static visualisations. It was completed as individual coursework for COMP4037 (Research Methods) at the University of Nottingham.

## Key Finding
A notable anomaly was identified in the Mental and Behavioural disorders specialty — its recovery pattern following COVID-19 was distinctly different from all other medical specialties, suggesting a unique and prolonged impact on this category.

## Visualisations
- **Interactive diverging heatmap** built in Plotly — shows trends across all specialties over 26 years
- **Supporting bar chart** built in Matplotlib — highlights specific comparisons

## How to Run
1. Clone this repository
2. Open the `.ipynb` notebook in Google Colab
3. Run all cells — the interactive Plotly heatmap will render inline
4. Alternatively, open the `.html` file directly in any browser for the standalone interactive version
